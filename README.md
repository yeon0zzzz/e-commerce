#### Running Docker Containers

`local` profile 로 실행하기 위하여 인프라가 설정되어 있는 Docker 컨테이너를 실행해주셔야 합니다.

```bash
docker-compose up -d
```

# 🛒 e-커머스 상품 주문 서비스
- 상품 주문에 필요한 메뉴 구성 및 조회 가능
- 상품 여러개 선택하여 주문 가능하며, 미리 충전한 잔액을 이용
- 상품 주문 내역을 통해 판매량 가장 높은 상품 추천
